# Personal Website

Portfolio website for Mohamed Fhafah, focused on cybersecurity, secure software engineering, and networked systems.

## Highlights

- React 19 + TypeScript + Vite
- Tailwind-based design system
- Animated hero canvas with lightweight 2D rendering
- Multilingual content (English, French, Arabic)
- Structured project, experience, education, and blog data
- CV download and direct contact CTA

## Project structure

```text
src/
├── components/
├── data/
├── i18n/
├── pages/
├── sections/
└── components/three/
```

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Verification

Use the production build as the primary health check:

```bash
npm run build
```

## Purpose

This repository is the public portfolio codebase used to present academic work, cybersecurity experience, and selected engineering projects to recruiters and collaborators.
